# pro-01 — CSV viewer

A CSV viewer that loads a local file (via `<input type="file">`) and lets the user sort columns. Exercises the File API, in-memory parsing of moderately structured data, and table-row interaction. Designed to surface regressions on real-world data shapes — a kid-tier dream rarely manipulates structured data beyond a 4-element array, but a pro-tier one routinely does.
